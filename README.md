# bindo-exercise
Bindo exercise

## Installation
This project uses Grunt for automation. For more info, visit: [http://gruntjs.com](http://gruntjs.com)


Install project dependencies

```
npm install
```

To start the server:

```
grunt http-server:dev
```

You can now access it via:

```
http://0.0.0.0:8282/
```
